⚙️ Configuração GitHub

Tempo de Sincronização entre Dispositivos:

  • GitHub API: Imediato (quando salva com sucesso)
  • GitHub Pages: 1-10 minutos para propagar
  • Cache do navegador: Pode precisar de refresh (Ctrl+F5)

Como obter um token GitHub:

  1. Acesse GitHub → Settings → Developer settings
  2. Personal access tokens → Tokens (classic)
  3. Generate new token com permissão "repo"
  4. Cole o token aqui
